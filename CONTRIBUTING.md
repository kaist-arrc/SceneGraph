# Contributing

SceneGraph is currently a proposal repository. Contributions should make the
schema easier to discuss, test, or apply.

Good contributions include:

- clearer definitions for graph layers
- example scenes
- edge and node type proposals
- validation rules
- mapping notes for existing formats
- open questions and design tradeoffs

When proposing schema changes, include:

1. the problem the change solves
2. one or more examples
3. compatibility concerns
4. any unresolved questions

Keep the core schema small. Prefer extensions for domain-specific fields unless
the concept is broadly useful across many scene graph domains.
